
A simple (gradient-heavy) single pager using jQuery. Fully responsive, built on Sass,
and loaded with an assortment of pre-styled elements.

Mobile web app to calculate daily carb intake. 
A company have asked to create a mobile web app that will allow users who are diabetic to 
track their daily carbohydrate intake.
Uses HTML, PHP, SQL, JavaScript and JQuery.
